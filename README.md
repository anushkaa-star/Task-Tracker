# 📌 Task Tracker

A simple Task Tracker built using HTML, CSS, and JavaScript. This project allows users to add tasks, assign deadlines, and delete tasks once they are completed. The goal of this project was to practice core JavaScript concepts while creating a clean and user-friendly interface.

## Features

* Add new tasks
* Set a deadline for each task
* Delete tasks when they are no longer needed
* Dynamic task display using JavaScript
* Simple and responsive user interface
* Background image and custom styling for a better user experience

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## What I Learned

While building this project, I gained hands-on experience with:

- JavaScript basics and functions
- Taking input from users
- Managing data using arrays
- Dynamically updating the webpage
- Creating structured layouts with CSS
- Building a clean and user-friendly interface

## How It Works

1. Enter a task in the input field.
2. Select a deadline using the date picker.
3. Click the **Add** button to save the task.
4. The task appears in the task list along with its deadline.
5. Click **Delete** to remove a task from the list.
